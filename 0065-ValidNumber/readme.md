# Valid Number

## Description

&emsp;&emsp;Validate if a given string is numeric.
            
## Example:

### eg1

```
    "0" => true
    
    " 0.1 " => true
    
    "abc" => false
    
    "1 a" => false
    
    "2e10" => true
```
