# Longest Palindromic Substring

## Description

&emsp;&emsp;Given a string **s**, find the longest palindromic substring in **s**. You may assume that the maximum 
length of **s** is 1000.

## Example:

### eg1

``` 
    Input: "babad"
    
    Output: "bab"
    
    Note: "aba" is also a valid answer.
```

### eg2

``` 
    Input: "cbbd"
    
    Output: "bb"
```
