# First Missing Positive

## Description

&emsp;&emsp;Given an unsorted integer array, find the smallest missing positive integer.

## Example:

### eg1

```
    Input: [1,2,0]
    
    Output: 3
```

### eg2

```
    Input: [3,4,-1,1]
    
    Output: 2
```

### eg3

```
    Input: [7,8,9,11,12]
    
    Output: 1
```

## Note

&emsp;&emsp;Your algorithm should run in O\(n\) time and uses constant extra space.

## Difficulty

&emsp;&emsp;Hard
