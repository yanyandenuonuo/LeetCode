# Sort List

## Description

&emsp;&emsp;Sort a linked list in O\(n log n\) time using constant space complexity.

## Example

### eg1

```
    Input: 4->2->1->3
    Output: 1->2->3->4
```

### eg2

```
    Input: -1->5->3->4->0
    Output: -1->0->3->4->5
```
