# Remove Duplicates from Sorted List

## Description

&emsp;&emsp;Given a sorted linked list, delete all duplicates such that each element appear only once.
            
## Example

### eg1

```
    Input: 1->1->2
    
    Output: 1->2
```

### eg2

```
    Input: 1->1->2->3->3
    
    Output: 1->2->3
```

&emsp;&emsp;Easy
