# Remove Duplicates from Sorted List II

## Description

&emsp;&emsp;Given a sorted linked list, delete all nodes that have duplicate numbers, leaving only distinct numbers 
from the original list.
            
## Example

### eg1

```
    Input: 1->2->3->3->4->4->5
    
    Output: 1->2->5
```

### eg2

```
    Input: 1->1->1->2->3
    
    Output: 2->3
```
