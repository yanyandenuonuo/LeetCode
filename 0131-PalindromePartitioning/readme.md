# Palindrome Partitioning

## Description

&emsp;&emsp;Given a string s, partition s such that every substring of the partition is a palindrome.

&emsp;&emsp;Return all possible palindrome partitioning of s.

## Example

### eg1

```
    Input: "aab"
    
    Output:
    [
        ["aa","b"],
        ["a","a","b"]
    ]
```
