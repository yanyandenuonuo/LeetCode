# Same Tree

## Description

&emsp;&emsp;Given two binary trees, write a function to check if they are the same or not.

&emsp;&emsp;Two binary trees are considered the same if they are structurally identical and the nodes have the same 
value.

## Example

### eg1

```
    Input:     1         1
              / \       / \
             2   3     2   3
    
            [1,2,3],   [1,2,3]
    
    Output: true
```

### eg2

```
    Input:     1         1
              /           \
             2             2
    
            [1,2],     [1,null,2]
    
    Output: false
```

### eg3

```
    Input:     1         1
              / \       / \
             2   1     1   2
    
            [1,2,1],   [1,1,2]
    
    Output: false
```
