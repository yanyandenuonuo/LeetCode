# Flatten Binary Tree to Linked List

## Description

&emsp;&emsp;Given a binary tree, flatten it to a linked list in-place.

## Example

### eg1

&emsp;&emsp;For example, given the following tree:

```
        1
       / \
      2   5
     / \   \
    3   4   6
```

&emsp;&emsp;The flattened tree should look like:

```
    1
     \
      2
       \
        3
         \
          4
           \
            5
             \
              6
```
