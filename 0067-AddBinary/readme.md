# Add Binary

## Description

&emsp;&emsp;Given two binary strings, return their sum (also a binary string).
            
&emsp;&emsp;The input strings are both **non-empty** and contains only characters **1** or **0**.
            
## Example:

### eg1

```
    Input: a = "11", b = "1"
    
    Output: "100"
```

### eg2

```
    Input: a = "1010", b = "1011"
    
    Output: "10101"
```

## Difficulty

&emsp;&emsp;Easy
