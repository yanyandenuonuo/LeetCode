# Gray Code

## Description

&emsp;&emsp;Given a collection of integers that might contain duplicates, **nums**, return all possible subsets 
\(the power set\).

&emsp;&emsp;**Note:** The solution set must not contain duplicate subsets.

## Example

### eg1

```
    Input: [1,2,2]
    
    Output:
    [
        [2],
        [1],
        [1,2,2],
        [2,2],
        [1,2],
        []
    ]
```

## Difficulty

&emsp;&emsp;Medium
