# Binary Tree Preorder Traversal

## Description

&emsp;&emsp;Given a binary tree, return the preorder traversal of its nodes' values.

## Example

### eg1

```
    Input: [1,null,2,3]
       1
        \
         2
        /
       3
    
    Output: [1,2,3]
```

&emsp;&emsp;**Follow up:** Recursive solution is trivial, could you do it iteratively?
