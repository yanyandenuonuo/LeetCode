# Generate Parentheses

## Description

&emsp;&emsp;Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.
            
&emsp;&emsp;For example, given n = 3, a solution set is:

``` 
    [
        "((()))",
        "(()())",
        "(())()",
        "()(())",
        "()()()"
    ]
```
