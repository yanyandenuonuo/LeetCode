# Merge k Sorted Lists

## Description

&emsp;&emsp;Merge k sorted linked lists and return it as one sorted list. Analyze and describe its complexity.
            
## Example:

### eg1

``` 
    Input:
    [
      1->4->5,
      1->3->4,
      2->6
    ]
    
    Output: 1->1->2->3->4->4->5->6
```
