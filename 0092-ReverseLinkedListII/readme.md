# Reverse Linked List II

## Description

&emsp;&emsp;Reverse a linked list from position m to n. Do it in one-pass.

&emsp;&emsp;Note: 1 ≤ m ≤ n ≤ length of list.

## Example

### eg1

```
    Input: 1->2->3->4->5->NULL, m = 2, n = 4
    
    Output: 1->4->3->2->5->NULL
```

## Difficulty

&emsp;&emsp;Medium
