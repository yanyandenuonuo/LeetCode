# Permutations II

## Description

&emsp;&emsp;Given a collection of numbers that might contain duplicates, return all possible unique permutations.

## Example:

### eg1

```
    Input: [1,1,2]
    
    Output:
    [
        [1,1,2],
        [1,2,1],
        [2,1,1]
    ]
```
