# Subsets

## Description

&emsp;&emsp;Given a set of **distinct** integers, nums, return all possible subsets (the power set).

&emsp;&emsp;**Note:** The solution set must not contain duplicate subsets.
            
## Example

### eg1

```
    Input: nums = [1,2,3]
    
    Output:
    [
        [3],
        [1],
        [2],
        [1,2,3],
        [1,3],
        [2,3],
        [1,2],
        []
    ]
```
