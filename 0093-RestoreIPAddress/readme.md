# Restore IP Addresses

## Description

&emsp;&emsp;Given a string containing only digits, restore it by returning all possible valid IP address combinations.

## Example

### eg1

```
    Input: "25525511135"
    
    Output: ["255.255.11.135", "255.255.111.35"]
```
