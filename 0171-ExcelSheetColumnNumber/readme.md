# Excel Sheet Column Number

## Description

&emsp;&emsp;Given a column title as appear in an Excel sheet, return its corresponding column number.

&emsp;&emsp;For example:

```
    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28 
    ...
```

## Example

### eg1

```
    Input: "A"
    
    Output: 1
```

### eg2

```
    Input: "AB"
    
    Output: 28
```

### eg3

```
    Input: "ZY"
    
    Output: 701
```
